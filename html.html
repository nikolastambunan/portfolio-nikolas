Belajar HTML & CSS Dasar: Panduan Lengkap untuk Pemula

📅 15 Maret 2024 ⏱️ 20 menit baca 📖 Level: Pemula

HTML dan CSS adalah fondasi dari setiap website yang Anda lihat di internet. HTML bertugas membangun struktur, sementara CSS membuat tampilan menjadi menarik. Artikel ini akan membawa Anda dari nol hingga mampu membuat halaman web sendiri.

📌 Apa itu HTML dan CSS?

HTML (HyperText Markup Language) adalah bahasa markup yang membangun kerangka halaman web. CSS (Cascading Style Sheets) adalah bahasa yang mengatur tampilan: warna, ukuran, tata letak, dan efek visual.

1. Struktur Dasar HTML

Setiap dokumen HTML memiliki struktur yang sama. Berikut adalah template dasar yang wajib Anda ketahui:

<!DOCTYPE html>
<html lang="id">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Judul Halaman</title>
</head>
<body>
    <h1>Selamat Datang</h1>
    <p>Ini adalah paragraf pertama saya.</p>
</body>
</html>
💡 Penjelasan Singkat

<!DOCTYPE html> memberitahu browser bahwa ini adalah HTML5. <head> berisi metadata dan judul. <body> adalah tempat konten yang terlihat oleh pengguna.

3. CSS: Cara Menambahkan dan Sintaks Dasar

Ada tiga cara menambahkan CSS ke HTML: Inline, Internal, dan External.

Inline CSS 
<h1 style="color: blue;">Teks Biru</h1>
Internal CSS 
<style>
    h1 { color: blue; }
</style>
<!-- External CSS (yang paling direkomendasikan) -->
<link rel="stylesheet" href="style.css">

/* File style.css */
h1 {
    color: blue;
    font-size: 32px;
    text-align: center;
}

Selektor CSS yang Wajib Diketahui

SelektorContohPenjelasan
Elemenp { }Memilih semua tag <p>
Class.card { }Memilih elemen dengan class="card"
ID#header { }Memilih elemen dengan id="header" (unik)
Universal* { }Memilih semua elemen

5. Flexbox: Layout Modern yang Wajib Dikuasai

Flexbox menyederhanakan pembuatan layout responsif. Berikut contoh-contoh praktis:

Flexbox Dasar

.container {
    display: flex;
    justify-content: space-between; /* horizontal: start, center, space-between */
    align-items: center;            /* vertical: center, flex-start, flex-end */
    gap: 20px;                      /* jarak antar item */
    flex-wrap: wrap;                /* membungkus jika layar kecil */
}

.item {
    flex: 1;      /* semua item punya lebar sama */
    padding: 16px;
    background: #eef2ff;
}

Contoh Navigasi dengan Flexbox

<!-- HTML -->
<nav class="navbar">
    <div class="logo">MySite</div>
    <ul class="nav-links">
        <li><a href="#">Beranda</a></li>
        <li><a href="#">Tentang</a></li>
        <li><a href="#">Kontak</a></li>
    </ul>
</nav>

/* CSS */
.navbar {
    display: flex;
    justify-content: space-between;
    align-items: center;
    padding: 1rem 2rem;
    background: #1f2937;
    color: white;
}

.nav-links {
    display: flex;
    gap: 1.5rem;
    list-style: none;
}

.nav-links a {
    color: white;
    text-decoration: none;
}
💡 Flexbox Properties yang Sering Dipakai

flex-direction: row | column — arah utama. justify-content — alignment horizontal. align-items — alignment vertikal. gap — jarak antar elemen.

7. Responsive Design dengan Media Query

Agar website tampil baik di semua perangkat (HP, tablet, desktop), gunakan media query.

/* Default untuk desktop */
.container {
    display: flex;
    gap: 2rem;
}

/* Tablet: lebar kurang dari 768px */
@media (max-width: 768px) {
    .container {
        flex-direction: column;
    }
}

/* HP: lebar kurang dari 480px */
@media (max-width: 480px) {
    body {
        font-size: 14px;
    }
    .container {
        padding: 1rem;
    }
}
📱 Tips Responsive

Gunakan unit relatif seperti %, rem, vw, vh daripada px untuk ukuran yang fleksibel. flex-wrap: wrap dan grid juga sangat membantu.

8. Best Practices untuk Pemula

  • Gunakan struktur folder yang rapi — pisahkan folder css/, images/, js/
  • Beri nama class yang bermakna — seperti .card, .nav-menu, bukan .box1
  • Gunakan CSS Reset — untuk menyamakan tampilan di semua browser
  • Validasi kode — gunakan W3C Validator untuk memastikan tidak ada error
  • Praktikkan dengan proyek kecil — landing page, portofolio, atau blog sederhana
⚠️ Kesalahan Umum Pemula

1. Lupa menutup tag HTML. 2. Tidak menggunakan alt pada gambar. 3. Menggunakan ID berlebihan (lebih baik class). 4. Tidak responsive.
